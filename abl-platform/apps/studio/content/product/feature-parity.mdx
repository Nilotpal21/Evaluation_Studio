## Overview

Agent Platform 2.0 is a ground-up rebuild, not a migration. While it covers all critical v1 capabilities, several features take different architectural approaches. See the [Feature Matrix](/docs/product/feature-matrix) for the complete breakdown.

## Parity Status

| Category         | v1 Features               | v2 Status       | Notes                                    |
| ---------------- | ------------------------- | --------------- | ---------------------------------------- |
| Agent Definition | Dialog tasks, sub-intents | **Complete**    | ABL replaces XO dialog builder           |
| NLU              | ML/KG/FM engines          | **Complete**    | Unified NLU with LLM classification      |
| Channels         | 30+ channels              | **In Progress** | 20+ channels live, remainder in pipeline |
| Knowledge AI     | FAQ, Articles             | **Complete**    | SearchAI with RAG, hybrid search         |
| Analytics        | Custom dashboards         | **In Progress** | ClickHouse pipeline, custom events       |
| Admin            | Full console              | **Complete**    | Workspace, RBAC, LLM credentials         |
| APIs             | Bot Builder API           | **Complete**    | REST + WebSocket + SDK                   |
| Enterprise       | SSO, Data residency       | **Partial**     | SSO complete, data residency roadmap     |
