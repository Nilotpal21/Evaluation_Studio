# Platform Feature Index

> **Last Updated**: 2026-04-22
> **SDLC Pipeline Status**: 84/101 core platform features have completed the full 4-phase pipeline (feature-spec → test-spec → HLD → LLD) with audit loops. Across the separate ancillary track, 1/2 docs are in progress and 1/2 remains at feature-spec only.

Comprehensive documentation for the ABL platform's major feature modules, focused sub-features, and cross-feature reference hubs. Each feature file follows the standard template (`TEMPLATE.md`) with sections covering overview, goals, user stories, numbered requirements, lifecycle impact, integration touchpoints, non-functional concerns, delivery tasks, and testing.

## Documentation Structure

- Major platform capabilities live at `docs/features/<feature>.md`.
- Focused docs that sit under a larger capability live at `docs/features/sub-features/<feature>.md`.
- Cross-feature overview hubs stay at the top level and link out to the dedicated major-feature docs.
- Matching testing guides mirror the same structure under `docs/testing/`.
