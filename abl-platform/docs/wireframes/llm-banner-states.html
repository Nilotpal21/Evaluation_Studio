Scenarios

Standard OpenAI Setup

Customer adds OpenAI API key, uses all defaults

→ Banner: "Connected to gpt-4o"

Anthropic Provider

Customer adds Anthropic API key, uses all defaults

→ Banner: "Connected to claude-sonnet-4-20250514" (or highest tier)

All Fast Tier Override

Customer overrides every use case to fast tier

→ Banner: "Connected to gpt-4o-mini"

No LLM Configured

New KB, no provider credentials set

→ Banner: "⚠️ Search will use keyword matching only..."
