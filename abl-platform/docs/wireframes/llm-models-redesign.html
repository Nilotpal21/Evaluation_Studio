Interaction Behavior

Toggle (below description): Enable/disable the feature. When ON, model auto-resolves and model info line appears. When OFF, model info and Change button disappear — only name + description + "Disabled" badge + toggle remain.

Change button (top-right): Opens the Model Selector Dialog (same as Query Pipeline LLM). Auto-select toggle is inside the dialog — not visible on the card. Only shown when feature is enabled.

Green colour / status styling: Active features keep the green status badge. Card colours and styling remain the same as current design.

No "auto-selected" badge on card: Removed to save space. The auto-select state is managed inside the dialog only. The status badge (Active/Fallback/Disabled) is the only badge on the card.
