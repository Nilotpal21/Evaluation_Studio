How this prototype works

1. Display Name (top) — type what you want to call the field, e.g. Urgency, Priority

2. Auto-suggest — as you type Display Name, Field Name automatically fills with the best matching connector field (green chip). Type, Description, and Values auto-fill too.

3. Not satisfied? — click the × on the green chip to clear it, then click Field Name dropdown to browse/search all 78 fields and pick a different one.

4. Type filter — select a Type first (e.g., Number), then the dropdown only shows fields of that type.

Try: type Urgency → auto-suggests "Urgency" field. Type Prior → auto-suggests "Priority". Type xyz → no match, pick manually from dropdown.
