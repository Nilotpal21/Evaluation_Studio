Queue Name Worker Purpose Retry Config Dependencies
ingestion ingestion-worker Entry point: creates SearchDocument, uploads to S3, routes to extraction queue by MIME type 3 attempts, exponential backoff S3, MongoDB
docling-extraction extraction-worker (Docling path) Sends document to Docling HTTP service for structured extraction (PDF, DOCX, HTML, PPTX) 3 attempts, 120s timeout, exponential backoff Docling service (port 8080)
search-extraction extraction-worker (legacy path) Direct text parsing for TXT, MD, CSV files without Docling 3 attempts, exponential backoff S3
page-processing page-processing-worker Applies chunking strategy (fixed/semantic/sliding/page). Optional summarization and question generation 3 attempts, exponential backoff LLM (for summarization/QGen)
canonical-map canonical-map-worker Builds canonical field mappings and syncs schema for structured metadata 2 attempts MongoDB
enrichment enrichment-worker Fan-out: dispatches to multimodal, tree-builder, scope-classifier. Text stats, language detection 2 attempts, exponential backoff LLM (optional)
visual-enrichment visual-enrichment-worker LLM vision describes images in document pages 2 attempts LLM with vision
multimodal multimodal-worker LLM vision processes tables and diagrams in chunks 2 attempts LLM with vision
embedding embedding-worker Generates vector embeddings via resolved provider (BGE-M3, OpenAI, etc.). Batch processing 3 attempts, exponential backoff BGE-M3 (port 8000) or OpenAI API
connector-sync sync-worker Full/delta sync for connectors (SharePoint, etc.). Distributed lock prevents duplicates 3 attempts, long lockDuration Provider API, Redis (lock), MongoDB
permission-crawl permission-crawl-worker Crawls file-level ACL permissions from connector providers 2 attempts Provider API, MongoDB
content-processing content-processing-worker Processes crawler output: readability extraction, S3 upload 3 attempts S3
crawler-ingestion crawler-ingestion-worker Creates SearchDocuments from crawled content and routes into main pipeline 3 attempts Go crawler service, S3
