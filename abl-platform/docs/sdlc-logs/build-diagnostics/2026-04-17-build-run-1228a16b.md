## Root Cause of Self-Reference Bug (FIXED 2026-04-17)

**Regression introduced by commit `f2e48bdcf` (2026-04-16 18:24)**

The commit added `if (agentSpec.isEntry && !context.plan)` — gating the explicit
`## Entry-Point Routing — CRITICAL` section behind `!context.plan`. Once
`computeArchitecturePlans` was wired in commit `72e01be01` (same day), `context.plan` is
always defined for every agent, so this CRITICAL section was silently skipped for all
entry/supervisor agents.

**Two compounding issues:**

1. `condition: undefined` in `build-parallel-gen.ts:1505` stripped WHEN conditions from the
   planner input, so `renderArchitecturePlan` rendered HANDOFF targets with no WHEN text
2. The plan HANDOFF format was comma-separated (`- TO: OrderSupport, type: delegate, RETURN: true`)
   — not ABL YAML — making it ambiguous for small models (Haiku) to translate to generated code

**Why the old code never had this bug:**

- The old `## Entry-Point Routing — CRITICAL` section used ABL-like syntax the LLM could copy:
  `- TO: OrderSupport WHEN: "When intent is order status..." RETURN: true`
- It was a top-level `##` section labeled CRITICAL — high attention weight for small models

**Why Haiku generated self-references specifically:**

- The LLM correctly read the WHEN conditions from the `## Topology Context` section
- But without a clear ABL-format mapping of WHEN → TO, it defaulted to using the currently-being-generated agent name (`SupportTriage`) as the TO target for all routes
- This is a small-model cross-section reasoning failure, not a prompt understanding failure

**Fix applied (2026-04-17):**

1. `handbook-reference.ts`: Removed `!context.plan` guard — `## Entry-Point Routing — CRITICAL`
   now always renders for entry/supervisor agents (plan is complementary, not a replacement)
2. `build-parallel-gen.ts`: Changed `condition: undefined` to `condition: e.condition` so the
   planner receives actual WHEN conditions from the topology edges
3. `handbook-reference.ts`: Changed `renderArchitecturePlan` HANDOFF section to emit ABL YAML
   format with WHEN conditions instead of comma-separated descriptors
